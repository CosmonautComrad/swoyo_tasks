     Цель этой работы заключалась в том, чтобы исследовать подходы для диагностики экспериментов лазерной физики с помощью методов машинного обучения на примере двух задач: определение параметров лазер-плазменного взаимодействия и решение обратной задачи для корректировки фокусировки лазерных импульсов.
Для решения первой задачи требовалось выполнить следующие шаги:
1.Обучить полносвязную нейронную сеть и оценить качество работы модели.
2.Рассмотреть классический метод понижения размерности – метод главных компонент. Проанализировать критерии оценки числа главных компонент.
3.Исследовать методы аугментации данных для улучшения точности восстановления параметров.
4.Проанализировать влияние шума, который может быть в экспериментальных данных, на точность модели. Предложить робастные подходы к оценки восстанавливаемых параметров с шумом в данных.
     Для решения второй задачи требовалось выполнить следующие шаги:
1.Предложить корректною постановку задачи, в которой будут восстанавливаться все рассматриваемые параметры.
2.Исследовать изменение точности модели при обучении на одном множестве фиксированных скрытых параметров и тестировании на другом множестве параметров.
